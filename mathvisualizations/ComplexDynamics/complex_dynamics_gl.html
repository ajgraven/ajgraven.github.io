Instructions

  • The left plot is the parameter space and the right is the dynamical plane.
  • The blue point in the parameter space plot can be dragged to change the value of c used in the dynamical plane plot.
  • The blue point in the dynamical plane is the starting point for a plot of the first 8 iterates, and can also be dragged.
  • The arrows and +/- buttons below each plot may be used to translate and zoom in/out the region in view.
  • The text boxes below the plots may be used to edit the function to be iterated, as well as the escape condition. (there are separate escape condition boxes for the dynamical plane and the parameter space to ameliorate problems of numerical stability in certain examples)
